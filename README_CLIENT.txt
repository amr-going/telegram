
Как пользоваться:
1. Запустите бота.
2. Введите пароль.
3. Используйте меню:
    - Сохранить файл: отправьте любой файл, нажмите "Готово"
    - Просмотр файлов: бот пришлёт список
    - Удаление: укажите номер
4. Через 5 минут бездействия потребуется повторный вход.
5. Чтобы сменить пароль — введите аварийный код в разделе "Настройки".
